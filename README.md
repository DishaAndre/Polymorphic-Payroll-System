# Polymorphic Payroll System
A simple C++ payroll management system built to practice Object-Oriented Programming. 
It’s a simple system that helps manage employees, calculate their salaries, and show a final report.

## What it does
- It lets you add two types of employees: **Hourly** (based on hours worked) and **Salaried**.
- It calculates salaries automatically using a concept called **Polymorphism**.

## How to use it
1. Make sure you have a C++ compiler installed (like `g++`).
2. Download `main.cpp` and save it to your computer.
3. Open your terminal in the folder where you saved the file.
4. Type this command to compile the code:
   ```bash
   g++ main.cpp -o payroll
